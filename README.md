# :camera:*QR-Code Scanner*
________
:iphone: **QR-Code Scanner** - *Это приложение, которое поможет вам в изучении геометрии. С помощью него вы можете отсканировать QR-код в учебнике, и у вас появится 3D-изображение фигуры, что облегчит понимание предмета.*

_____
# :pushpin:*Основные возможности*
+ :white_check_mark: Сканирование QR-кодов 
+ :white_check_mark: Создание 3D фигур 
+ :white_check_mark: Осмотор фигуры со всех сторон 
_____
